# 系统循环逻辑流程

> 返回 [文档索引](./README.md)

---

## 主循环流程

```
1. 学校学习
   出席科目 → 完成次数累积 → 解锁职业
   ├── 小学：4 次完成一门科目
   ├── 中学：8 次完成一门科目
   └── 大学：16 次完成一门科目

2. 工作赚钱
   玩家让宠物工作 → 工资获得 → 健康值下降

3. 生病判定
   健康值 < 50 → 生病概率判定 → 初级疾病阶段

4. 治疗恢复
   玩家选择药物 → 进入静养期 → 恢复健康

5. 疾病恶化（不治疗）
   不治疗 → 升级疾病阶段 → 静养时间递增 → 药物价格上升

6. 死亡判定
   晚期连续 3 天未治疗 → 死亡 → 玩家选择复活或重新养宠物

7. 小游戏调节
   小游戏可增加健康、快乐和金币，辅助管理数值

8. 持续循环
   玩家持续管理宠物状态 → 循环进行
```
