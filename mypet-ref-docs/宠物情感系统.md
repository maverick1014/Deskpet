# 宠物情感系统（AI 情感引擎）

> 返回 [文档索引](./README.md)

---

## 概述

每只宠物都是一个**微型 AI 个体**，拥有独立的情感状态和性格特征。通过 AI 情感引擎驱动，宠物会根据自身性格、当前情绪、历史经历做出不同的反应和行为，让每只宠物都是独一无二的。

---

## 性格系统

### 性格维度

每只宠物在创建时随机生成或由玩家选择性格倾向，性格影响宠物对事件的反应方式。

| 性格维度 | 低值表现 | 高值表现 | 影响范围 |
|----------|----------|----------|----------|
| 活泼度 | 安静、慵懒 | 好动、爱玩 | 小游戏意愿、快乐度变化速率 |
| 胆量 | 胆小、容易紧张 | 勇敢、抗压 | 生病时的情绪波动、工作意愿 |
| 亲密度倾向 | 独立、不粘人 | 粘人、需要陪伴 | 独处时快乐度下降速率 |
| 食欲 | 挑食、吃得少 | 贪吃、吃得多 | 饱腹度下降速率、食物偏好 |
| 好奇心 | 保守、安于现状 | 好奇、喜欢探索 | 学习效率、新事物接受度 |

### 性格生成规则

- 每个维度取值范围：0~100
- 创建宠物时随机生成基础性格值
- 性格会随长期互动**缓慢微调**（如经常玩游戏 → 活泼度缓慢上升）
- 性格变化幅度有上限，保持宠物核心个性稳定

## 情绪系统

### 情绪状态

宠物拥有实时情绪，由 AI 情感引擎根据当前数值、事件和性格综合计算。

| 情绪 | 触发条件示例 | 表现行为 |
|------|-------------|----------|
| 开心 | 快乐度高、刚玩完游戏、吃到喜欢的食物 | 蹦跳、撒娇、主动互动 |
| 平静 | 数值均衡、无特殊事件 | 正常待机动画 |
| 无聊 | 长时间无互动、快乐度偏低 | 打哈欠、东张西望、戳玩家 |
| 饥饿 | 饱腹度低 | 看食物、摸肚子、委屈表情 |
| 生气 | 连续强迫工作、忽视需求 | 背对玩家、拒绝互动 |
| 伤心 | 生病、快乐度极低、被忽视太久 | 流泪、蜷缩、低头 |
| 害怕 | 生重病、健康值极低 | 发抖、躲藏、求助表情 |
| 疲惫 | 连续工作、过劳 | 打瞌睡、动作变慢 |

### 情绪计算逻辑

```
当前情绪 = f(基础数值, 性格, 近期事件, 情绪惯性)
```

- **基础数值权重**：健康值、饱腹度、清洁度、快乐度直接影响情绪基调
- **性格调节**：同样的数值，不同性格的宠物反应不同（如胆小的宠物生病时更害怕）
- **近期事件**：最近发生的事件有短期情绪加成/减益
- **情绪惯性**：情绪不会瞬间剧变，有平滑过渡

## AI 情感引擎

### 引擎定位

每只宠物内部运行一个轻量级 AI 决策模型，使宠物表现出"有灵魂"的行为。

### 核心能力

| 能力 | 说明 |
|------|------|
| 情绪推理 | 根据多维输入推断当前情绪状态 |
| 行为决策 | 基于情绪和性格选择待机行为/主动行为 |
| 对话/表达 | 通过文字气泡、表情、动作表达当前状态 |
| 记忆系统 | 记录玩家互动模式，形成长期偏好和信任度 |
| 个性化反应 | 同一事件，不同宠物有不同反应 |

### 输入参数

```
AI 引擎输入:
├── 当前数值（健康/饱腹/清洁/快乐）
├── 性格参数（5 维度）
├── 疾病状态
├── 工作/学习状态
├── 近期事件队列（最近 N 次互动记录）
├── 长期记忆（玩家互动模式摘要）
└── 时间因素（时段、距上次互动时长）
```

### 输出行为

```
AI 引擎输出:
├── 情绪标签（开心/伤心/生气/...）
├── 情绪强度（0~100）
├── 推荐动画/表情
├── 对话内容（文字气泡）
├── 主动行为（如主动找玩家、拒绝工作等）
└── 隐藏参数更新（信任度、亲密度等）
```

### 实现方案选项

| 方案 | 优点 | 缺点 | 适用阶段 |
|------|------|------|----------|
| 规则引擎（状态机 + 权重表） | 轻量、可控、离线运行 | 行为模式固定 | Beta / MVP |
| 本地轻量模型（TFLite） | 离线、个性化 | 模型体积、训练成本 | 后续迭代 |
| 云端 LLM API（如 GPT） | 对话自然、行为丰富 | 依赖网络、成本高 | 高级功能 |
| 混合方案 | 离线基础 + 云端增强 | 复杂度高 | 长期目标 |

> **Beta 1 建议**：先用规则引擎实现基础情感逻辑，预留 AI 模型接口，后续逐步替换为更智能的方案。

## 记忆系统

### 短期记忆

- 存储最近 N 次互动事件（如：喂食、游戏、工作、治疗）
- 影响即时情绪反应
- 过期后自动清除

### 长期记忆

- 统计玩家互动模式（如：经常忘记喂食、喜欢玩游戏、经常强迫工作）
- 影响宠物对玩家的信任度和亲密度
- 形成宠物的"对玩家的印象"

### 信任度

| 信任等级 | 行为表现 |
|----------|----------|
| 高信任 | 主动撒娇、听话、配合工作 |
| 中等信任 | 正常互动 |
| 低信任 | 偶尔拒绝指令、闹脾气 |
| 极低信任 | 经常拒绝、背对玩家、离家出走事件 |

## 数据存储

| 数据类型 | 存储方式 |
|----------|----------|
| 性格参数 | SQLite（宠物创建时生成，缓慢变化） |
| 当前情绪 | 内存（实时计算，不持久化） |
| 短期记忆 | SQLite（事件队列，定期清理） |
| 长期记忆 | SQLite（玩家互动模式统计） |
| 信任度 | SQLite（长期累积值） |
