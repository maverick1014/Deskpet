# 疾病系统

> 返回 [文档索引](./README.md)

---

## 触发条件

- 健康值 < 50 才有生病概率
- 健康值 ≥ 50 时不生病
- 生病阶段从【初级】开始

---

## 疾病阶段

### 初级（轻症）

| 项目 | 数值 |
|------|------|
| 药价 | 20G |
| 静养时间 | 2 小时 |
| 工作效率 | -10% |
| 静养期间恢复 | 每小时 +15 健康 |
| 不治疗后果 | 升级为中级 |

- 静养期间不可工作

### 中级（中症）

| 项目 | 数值 |
|------|------|
| 药价 | 35G |
| 静养时间 | 4 小时 |
| 工作效率 | -25% |
| 健康值持续下降 | -5/天 |
| 静养期间恢复 | 每小时 +15 健康 |
| 不治疗后果 | 升级为晚期 |

- 静养期间不可工作

### 晚期（重症）

| 项目 | 数值 |
|------|------|
| 药价 | 60G |
| 静养时间 | 6 小时 |
| 工作效率 | -50% |
| 健康值持续下降 | -10/天 |
| 静养期间恢复 | 每小时 +15 健康 |
| 不治疗后果 | 连续 3 天未治疗 → 死亡 |

- 静养期间不可工作

## 静养期机制

- 不可工作
- 按疾病阶段时间递增：初级 2h、中级 4h、晚期 6h
- 每小时 +15 健康
- 吃药后进入静养期，结束后痊愈

---

## 死亡机制

- 连续 3 天晚期未治疗 → 死亡
- 玩家可选择：
  - **复活**：复活丹 400G，健康恢复 60，虚弱 1 天（工作效率 -30%）
  - **重新养宠物**：新宠物从头开始养育
