# 精灵图规范（给动画师参考）

> 返回 [文档索引](./README.md)

---

## 概述

本文档定义了桌宠所有动画的精灵图（Spritesheet）规范。  
动画师请按照以下规格制作每组动画帧，完成后放入 `assets/sprites/` 目录。

---

## 通用规格

| 项目 | 规格 |
|------|------|
| 单帧尺寸 | 128 × 128 px（建议，可协商） |
| 图片格式 | PNG（透明背景） |
| 排列方式 | 水平排列，每行一个动画状态 |
| 命名规则 | `pet_{动画状态名}.png`（如 `pet_idle.png`） |
| 朝向 | 默认面朝**右**，程序中镜像翻转实现朝左 |

---

### 🟢 Stage 1 必须

| 状态 ID | 动画名称 | 帧数 | 帧率 (fps) | 循环 | 说明 |

| `idle_sleepy` | 待机犯困 | 4~6 帧 | 3 | ✅ 循环 | 打瞌睡 + zzZ |
| `walk` | 走路 | 6~8 帧 | 8 | ✅ 循环 | 正常步行（面朝右） |
| `run` | 跑步 | 6~8 帧 | 12 | ✅ 循环 | 快速跑动 |
| `climb_up` | 向上爬 | 4~6 帧 | 6 | ✅ 循环 | 爬屏幕边缘向上 |

## 注意事项

1. 所有帧的**锚点（脚底）**保持一致，避免行走/待机切换时角色上下跳动
2. 默认面朝**右**，程序中通过 `Canvas.scale(-1, 1)` 镜像翻转
3. 透明背景 PNG，不要有背景色
4. 情绪动画（happy/sad/angry）和基础动画（idle/walk）尽量保持角色体型一致
5. 如果帧数有变化，请同步更新 `lib/pet/sprite_config.dart` 中的配置
