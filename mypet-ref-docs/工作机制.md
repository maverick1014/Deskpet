## 工资标准

按学历分类，每小时工资 = 现实工资 × 2 倍

### 小学学历职业

| 职业 | 时薪 |
|------|------|
| 清洁工 | 16G/h |
| 搬运工 | 18G/h |
| 农夫 | 20G/h |

| 快递员 | 28G/h |

### 大学学历职业

| 职业 | 时薪 |
|------|------|
| 白领 | 40G/h |
| 程序员 | 50G/h |
| 设计师 | 46G/h |

## 工作规则

- 每个学历可选择多种职业（至少三种）
- 工作时间：30 分钟/次
- 连续工作 5 次以上才触发过劳判定
- 工作期间不生病
- 完成工作后触发健康值下降及生病概率判定
