# 食物与药物

> 返回 [文档索引](./README.md)

---

## 食物

- 价格：5G 起
- 效果：增加健康值 / 饱腹度

## 药物

- 价格：20G 起
- 使用条件：仅在生病状态下可使用
- 效果：不会立即恢复，使用后进入静养期
- 静养时间根据疾病等级不同而递增

## 药物价格表

| 疾病阶段 | 药价 |
|----------|------|
| 初级（轻症） | 20G |
| 中级（中症） | 35G |
| 晚期（重症） | 60G |
